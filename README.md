# 📊 Telecom X LATAM - Churn Analysis

## 🎯 Objetivo
Analizar los factores asociados a la cancelación de clientes (Churn) para generar insights estratégicos orientados a la retención.

## 🛠 Herramientas Utilizadas
- Python
- Pandas
- Matplotlib
- Requests (Consumo de API)
- Google Colab

## 📈 Proceso de Análisis
1. Extracción de datos desde API
2. Limpieza y transformación
3. Análisis exploratorio (EDA)
4. Identificación de variables asociadas al churn
5. Recomendaciones estratégicas

## 🔎 Principales Hallazgos
- Mayor tasa de cancelación en contratos mensuales.
- Clientes con baja antigüedad presentan mayor riesgo.
- Impacto directo en ingresos recurrentes.

## 💡 Conclusión
El análisis permitió identificar segmentos críticos de clientes y proponer estrategias de retención basadas en datos.
